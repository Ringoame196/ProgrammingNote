## String -> 分に変換

```java
public static int toMinutes(String time) {
  DateTimeFormatter formatter = DateTimeFormatter.ofPattern("HH:mm"); // この形式の文字列を日時として扱うよ
  LocalTime localTime = LocalTime.parse(time, formatter); // 受け取った文字列を LocalTime に変換する

    return localTime.getHour() * 60 + localTime.getMinute(); // 合計分を計算し返す
}
```
