# ブランチの作成と変更

## 新しいブランチを作成し、すぐにそのブランチに切り替えます
```sh
git checkout -b <新しいブランチ名>
```

※ Git 2.23以降（推奨）
```sh
git switch -c <新しいブランチ名>
```

## 新しいブランチを作成します
```sh
git branch <新しいブランチ名>
```

## 既存のブランチに切り替えます
```sh
git checkout <ブランチ名>
```

※ Git 2.23以降（推奨）
```sh
git switch <ブランチ名>
```

## 現在存在するブランチ一覧を表示します
```sh
git branch
```

## リモートブランチも含めて一覧表示します
```sh
git branch -a
```

## 現在のブランチと変更状況を確認します
```sh
git status
```

## ブランチ名を変更します
```sh
git branch -m <古いブランチ名> <新しいブランチ名>
```

## 不要になったブランチを削除します（マージ済み）
```sh
git branch -d <ブランチ名>
```

## 強制的にブランチを削除します（未マージ）
```sh
git branch -D <ブランチ名>
```

## リモートブランチを削除します
```sh
git push origin --delete <ブランチ名>
```

## リモートの最新ブランチ情報を取得します
```sh
git fetch
```

## 一時的に作業内容を退避します
```sh
git stash
```

### 退避した作業を元に戻します
```sh
git stash pop
```

## main ブランチを最新状態に更新します
```sh
git checkout main
git pull origin main
```
