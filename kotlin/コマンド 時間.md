## 定期実行

```kotlin
import org.bukkit.scheduler.BukkitRunnable

object Timer : BukkitRunnable() {

    override fun run() {
        
        }
    }
}

// Main.kt
Timer.runTaskTimer(plugin, 0L, 20L)
```

## 1秒後実行
```kotlin
Bukkit.getScheduler().runTaskLater(plugin, Runnable { // 1秒後に実行したいコードをここに書く

}, 20L) // 20Lは1秒を表す（1秒 = 20ticks）
```
