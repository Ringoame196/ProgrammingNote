## データ検索
```sql
SELECT 列名 [表示用列名] FROM テーブル名;
SELECT * FROM テーブル名;
```

## 重複削除
```sql
SELECT DISTINCT 列名 FROM テーブル名;
```

## 条件式
```sql
WHERE 条件式;
```

## データ追加
```sql
INSERT INTO テーブル名 VALUES (値, 値, ...);
```

## データ更新
```sql
UPDATE テーブル名 SET 列名 = 値 WHERE 条件式;
```

## データ削除
```sql
DELETE FROM テーブル名 WHERE 条件式;
```
