# 平均（AVG）関数について
## AVG関数とは
`AVG()` は、指定した列の **平均値（Arithmetic Mean）** を求める集約関数（集計関数）です。

### 基本構文
```sql
SELECT AVG(カラム名)
FROM テーブル名;
```

### 使用例
```sql
SELECT AVG(count)
FROM Items;
```
- このSQLは Items テーブルの count カラムの平均を計算します。

## サブクエリでの使用例
平均以上の注文を取得する
```sql
WHERE 
    i.count >= (
        SELECT AVG(count)
        FROM Items
    )
```
### ポイント解説
- AVG(count) の部分はサブクエリ（副問い合わせ）。
- FROM Items があるため、count がどのテーブルの列か明確。
- よって i.count のように別名をつける必要はない。
